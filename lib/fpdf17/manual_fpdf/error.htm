Error

Error(string msg)

Descrição

Este método é chamado automaticamente em caso de erro fatal; ele simplesmente mostra a mensagem e para a execução. Uma classe herdada pode cancelá-lo para personalizar a manipulação de erro mas deve sempre parar o script, ou o documento resultante provavelmente será inválido.

Parâmetros

msg
A mensagem de erro.
